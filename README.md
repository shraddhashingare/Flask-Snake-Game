# Flask Snake Game (Dockerized)

A simple **Snake Game** built with **Flask (Python) and containerized **.  
---

## 🐳 Docker Setup

### 1️⃣ Build Docker Image
```bash
docker build -t flask-snake-game .
```
### 2️⃣ Run Container
```bash
docker run -d -p 5000:5000 flask-snake-game
```
### 3️⃣ Access the Game
```bash
http://localhost:5000
```
